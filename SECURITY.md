# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in VAINOM, please do
NOT open a public issue.

Instead, report it privately by email to:

**hello@vainom.com**

Please include:

- A description of the vulnerability
- Steps to reproduce
- VAINOM version affected (Windows or Android)
- Your OS and hardware configuration
- Any proof-of-concept code, if applicable

You can expect an initial response within 7 days. Critical
issues will be prioritized.

## Scope

VAINOM is currently in public beta (v0.5.8). Security reports
are welcome for:

- The Windows desktop application (VAINOM.exe)
- The Android application (APK)

Out of scope:

- Third-party AI models downloaded by the user
- Issues in upstream dependencies (report to their projects)
- Social engineering attacks

## Supported Versions

Only the latest public release from vainom.com is supported.
Older beta builds will not receive security patches.

## Disclosure Policy

We follow a coordinated disclosure approach. Please allow us
reasonable time to release a fix before public disclosure.
